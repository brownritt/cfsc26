# cfsc26

Computational Fluency Short Course for Carney Summer Scholars, 2026

Materials for the Carney Institute's 2026 Computational Fluency Short Course. Much of the course content consists of hands-on exercises, in some cases using code in other repos. This repo contains lecture and reference information, updated throughout the course.
